## How to download the project:

```bash
# Clone the repository
$ git clone https://github.com/rodrigoaraujo7/ecommerce-cart-system.git

# Enter directory
$ cd ecommerce-cart-system
```

## Usage

1. Run `yarn jsondb`.
2. Run `yarn start and access http://localhost:3000/`.
